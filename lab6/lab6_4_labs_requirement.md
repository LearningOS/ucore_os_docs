## 实验报告要求

从 git server 网站上取得 ucore_lab 后，进入目录 labcodes/lab6，完成实验要求的各个练习。
在实验报告中回答所有练习中提出的问题。在目录 labcodes/lab6 下存放实验报告，实验报告文档命名为 lab6.md，使用**markdown**格式。
